Returns a reactive boolean indicating whether a flag is enabled.

## Usage

```vue
<template>
  <div v-if="showNewDashboard">
    <NewDashboard />
  </div>
  <div v-else>
    <LegacyDashboard />
  </div>
</template>

<script setup>
const showNewDashboard = useFlag('new-dashboard')
</script>
```

## Type

```ts
function useFlag(name: string): ComputedRef<boolean>
```

::field-group
  ::field{name="name" type="string" required}
  The feature flag name as defined in Unleash.
  ::
::

## Returns

`ComputedRef<boolean>` — `true` if the flag is enabled, `false` otherwise. Returns `false` for unknown flags.
