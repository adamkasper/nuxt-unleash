Returns a reactive record of all feature flags. Useful for debugging or bulk flag inspection.

## Usage

```vue
<template>
  <pre>{{ JSON.stringify(allFlags, null, 2) }}</pre>
</template>

<script setup>
const allFlags = useAllFlags()
</script>
```

## Type

```ts
function useAllFlags(): ComputedRef<Record<string, EvaluatedFlag>>

interface EvaluatedFlag {
  name: string
  enabled: boolean
  variant: Variant
}
```

## Returns

`ComputedRef<Record<string, EvaluatedFlag>>` — all flags keyed by name. Returns `{}` if no flags are loaded.
